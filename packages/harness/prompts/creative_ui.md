## STEP 5: Focus state as SIGNATURE element

The focus state is the most-seen UI element on TV. Make it extraordinary and UNIQUE to this app:

DON'T just use border+scale. Create a multi-layered focus treatment:

```typescript
const focusedCard = {
  transform: [{ scale: 1.08 }],
  borderWidth: scaledPixels(4),
  borderColor: accentColor,
  // THE GLOW — this is the signature
  shadowColor: accentColor,
  shadowOffset: { width: 0, height: 0 },
  shadowOpacity: 0.7,

```

## STEP 6: Screen-specific atmosphere

Each screen should feel slightly different while maintaining the same design language:

**HomeScreen**: The flagship — most dramatic. Full atmospheric background, hero with cinematic gradient overlays, generous spacing.

**Detail/Player screens**: More intimate — slightly tighter spacing, muted background (less atmospheric glow), content-focused.

Add variety by:

- Varying the accent glow position (top-right on home, bottom-left on detail)
- Adjusting surface opacity per screen context

## STEP 7: Rail and section polish

Section titles need character:
- Add a short accent-colored line (40-80px) below or beside section titles
- Use letter-spacing on section titles: `letterSpacing: scaledPixels(1)` or text-transform uppercase for captions
- Give sections generous vertical spacing (scaledPixels(48-64) between sections)

Card rails:
- Cards should have a subtle surface color (not flat background) — `backgroundColor: 'rgba(255,255,255,0.03)'` or surface-1 color
- Unfocused cards should have very subtle borders (1px, 10% white) for definition
- On dark backgrounds, cards without any border/shadow look like floating patches — add minimal definition
